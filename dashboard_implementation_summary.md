# 📊 JobGiga Employer Analytics Dashboard

The JobGiga Employer Analytics Dashboard has been successfully implemented using **Next.js 15+ (App Router)** and **shadcn/ui** with **Tailwind CSS 4**.

## ✨ Key Features

1.  **Overview Summary Cards**: Instant access to real-time KPIs using shadcn `Card` and `Badge` components.
2.  **Interactive Trends**: Dynamic line/area charts using `Recharts`, integrated with shadcn `Select` for period filtering.
3.  **Candidate Funnel**: A visual, staged pipeline built with shadcn's accessible component architecture.
4.  **Job Performance Grid**: Side-by-side comparison of job listings using shadcn `Table` and `Card`.
5.  **Visibility & Interview Activity**: Granular insights into job exposure, visualized through consistent shadcn modules.
6.  **AI Power (Smart Insights)**: Dedicated AI-powered recommendation engine using ultra-modern shadcn styling.

## 🎨 Design Philosophy (shadcn style)

- **Standardized Excellence**: Leveraging **shadcn/ui** for a world-class, accessible, and consistent component library.
- **Premium Aesthetics**: Clean, professional design with an **Enterprise SaaS** aesthetic.
- **Modern Typography**: Using **Outfit** for superior readability and a modern feel.
- **Shadow-Free System**: Sleek 12px border radius with clean borders, avoiding heavy shadows.
- **Brand Consistency**: Theme based on a sophisticated **Indigo HSL palette** (`oklch` standard).
- **Animations**: Purposeful micro-interactions with **Framer Motion**.

## 🛠️ Technical Stack

- **Framework**: Next.js 15+ (TypeScript)
- **UI Library**: shadcn/ui (Radix UI)
- **Styling**: Tailwind CSS 4 (OKLCH color system)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Motion**: Framer Motion

## 🚀 How to Run Locally

```bash
npm run dev
```

The application is fully responsive and optimized for production.
