# 📊 JobGiga Employer Analytics Dashboard

The JobGiga Employer Analytics Dashboard has been successfully implemented using **Next.js 15+ (App Router)** and **Tailwind CSS 4**.

## ✨ Key Features

1.  **Overview Summary Cards**: Instant access to real-time KPIs (Active Jobs, Total Applicants, Hires, Views).
2.  **Interactive Trends**: Dynamic line/area charts using `Recharts` to monitor application volume.
3.  **Candidate Funnel**: A visual, staged pipeline for identifying bottlenecks and tracking conversion.
4.  **Job Performance Grid**: Side-by-side comparison of job listings with comparative engagement data.
5.  **Visibility & Interview Activity**: Granular insights into job exposure and candidate engagement levels.
6.  **AI Power (Smart Insights)**: Dedicated AI-powered recommendation engine to optimize hiring strategy.

## 🎨 Design Philosophy

- **Premium Aesthetics**: Clean, professional, and modern dashboard experience.
- **Modern Typography**: Using **Outfit** for superior readability and a modern feel.
- **Shadow-Free System**: Enforcing a sleek 12px border radius with clean borders, avoiding heavy shadows as per project guidelines.
- **Brand Consistency**: Theme based on a sophisticated Indigo palette with emerald success accents.
- **Animations**: Subtle, purposeful entry animations using **Framer Motion** for a "wow" factor.

## 🛠️ Technical Stack

- **Framework**: Next.js 15+ with TypeScript.
- **Styling**: Tailwind CSS 4 with custom design tokens.
- **Charts**: Recharts for data-dense visualizations.
- **Icons**: Lucide React for consistent, high-quality iconography.
- **Motion**: Framer Motion for premium micro-interactions.

## 🚀 How to Run Locally

```bash
npm run dev
```

The application is fully responsive and optimized for production.
